# Sound Player — MMA Lab 1

A browser-based audio player built with plain HTML, CSS, and JavaScript using the Web Audio API.

## Features

- **Play / Pause** — load any audio file and control playback
- **Volume slider** — adjust volume with a live percentage display
- **Mute / Unmute** — toggle audio with a visual indicator
- **Loop** — repeat the current track
- **Playback speed** — range from 0.5x to 2x
- **Seek bar** — click or drag to jump to any point in the track
- **Current time display** — shows elapsed and total duration
- **Skip** — jump forward or backward 10 seconds
- **Keyboard shortcuts** — `Space` play/pause, `M` mute, `L` loop, `←` `→` skip

## Usage

1. Open `index.html` in a browser
2. Click **Choose Audio File** and select an `.mp3`, `.wav`, or similar file
3. Use the on-screen controls or keyboard shortcuts to control playback
